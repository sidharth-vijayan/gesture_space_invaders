# Gesture-Controlled Space Invaders

A Space-Invaders-style game controlled entirely with webcam gestures using MediaPipe Hands + Pygame.

---

## Features
- Move ship horizontally by moving your hand
- Pinch gesture (thumb + index) fires bullets
- Optional two-finger gesture can be mapped to shield
- Smoothing + deadzone to avoid jitter
- Live webcam preview overlay in game window
- Score, lives, level progression, enemy shots
- Local high score saved to home directory
- Tests for smoothing and deadzone logic
